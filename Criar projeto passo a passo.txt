>>> Criar a pasta com o Spring
1. Entrar no site https://start.spring.io/
2. Projeto: como "Gradle - Groovy"
3. Linguagem: Java
4. Spring Boot: A versão mais recente sem "SANPSHOT", "M1" ou "M2"
5. Projeto Metadata: Informações do projeto
6. Packaging: Jar
7. Java: Versão mais recente
8. Dependecies: Adicionar a dependência "Spring Web"
9. Gerar o projeto no botão "Generate"
